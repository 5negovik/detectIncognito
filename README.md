# detectIncognito.js
JS detection of Incognito & other private browsing modes on Chrome, Edge, Safari, Firefox and MSIE.

# Supported
 * Safari for iOS - 8 to 14
 * Safari for macOS <= 14
 * Chrome/Chromium - 50 to 94 Dev
 * Edge - 15 to 18; 79 to 94 Dev
 * Firefox - 44 to 92 Beta
 * MSIE >= 10
 * Brave
