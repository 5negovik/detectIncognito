# detectIncognito.js
JS detection of Incognito & other private browsing modes on Chrome, Edge, Safari, Firefox and MSIE.

# Supported
 * Safari for iOS - 8 to 15
 * Safari for macOS <= 15
 * Chrome/Chromium - 50 to 95 Dev
 * Edge - 15 to 18; 79 to 95 Dev
 * Firefox - 44 to 93 Beta
 * MSIE >= 10
 * Brave
